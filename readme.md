# 🛒 Mon Message Panier (MyCartInfo)

Ce module permet d'ajouter un véritable encart d'information dans la page panier de votre boutique PrestaShop.

## 📝 En savoir plus

L'affichage dans le panier d'achat se fait très simplement : 
Le module permet l'affichage d'un message aux clients depuis le Back-office. Ce texte supporte le HTML, ce qui permet d'insérer des liens, de mettre en gras, et d'organiser le contenu de l'alerte !

## 🚀 Bénéfices

- **Informez vos clients** d'un nouveau transporteur.
- **Avertissez** d'un retard possible selon certaines conditions.
- **Partagez** un code promo visible avant de payer.
- **Aidez vos clients** à valider leur panier.

## ✨ Fonctionnalités

- Personnalisation du message en texte riche (HTML)
- Sélecteur de couleur dynamique (Bleu, Vert, Orange, Rouge)
- Responsive et adapté aux mobiles (Grille Bootstrap 4)
